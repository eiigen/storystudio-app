# StoryStudio — AI Storybook Generator

Create illustrated children's stories with AI. Enter a theme, pick a text and image model, and StoryStudio writes a multi-page story with matching illustrations and optional audio narration.

Built with [Pollinations.ai](https://pollinations.ai) — text generation via chat completions, images via CDN, audio via TTS.

## Live

https://eiigen.github.io/storystudio-app/

## Usage

1. Enter your Pollinations API key (`sk_...`)
2. Type a story theme (e.g. "A dragon who loves baking cookies")
3. Choose text and image models
4. Pick page count (2–5)
5. Click Generate

Stories are saved in your browser (localStorage).

## Tech
